同一图样中，同类项的图线宽度应一致。虚线、点画线等的线段长和间隔应尽量一致。

绘制圆的轴线时，圆心应为线段的交线。点画线、双点划线的首尾应为线段，且超出轮廓线2-5mm。

在较小的图形上可以用细实线代替点画线。

当虚线与虚线，或虚线与粗实线相交，应该是线段相交。当虚线是粗实线的延长线时，在连线处应断开。

图形的投影

投影面分为三个：正立投影面（V面）、水平投影面（H面）、侧立投影面（W面），如下图所示。

Figure 三个投影面

根据直线与三个投影面的位置关系，可以将直线分为三种：投影面平行线、投影面垂直线和一般位置直线。

对于投影面平行线，平行于H面的称为 水平线 ，平行于V面的称为 正平线 ，平行于W面的称为 侧平线 。

对于投影面垂直线，垂直于H面的称为 铅垂线 ，垂直于V面的称为 正垂线 ，垂直于W面的称为 侧垂线 。

Figure 特殊位置直线的投影

根据平面与三个投影面的位置关系，可以将平面分为三种：投影面垂直面、投影面平行面和一般位置平面。

只垂直于一个投影面而与其他两个投影面都处于倾斜位置的平面称为投影面垂直面。其中，垂直于V面，同时与H面和W面处于倾斜位置的平面，称为 正垂面 ；垂直于H面，同时与V面和W面处于倾斜位置的平面，称为 铅垂面 ；垂直于W面，同时与H面和V面处于倾斜位置的平面，称为 侧垂面 。

平行于某一投影面的平面称为投影面平行面。其中，平行于V面的称为 正平面 ；平行于H面的称为 水平面 ；平行于W面的称为 侧平面 。

立体与立体相交

相贯线是由平面曲线或直线组成的空间折线，每一段是平面体表面与回转体表面的交线。求相贯线的实质是求平面体各面与回转面的截交线。

Figure 平面体与回转体相贯

回转体与回转体相贯的相贯线一般为光滑封闭的空间曲线，它是两回转体表面的共有线。

直径不相等的两正交圆柱相贯，相贯线在平行于两圆柱轴线的投影面上的投影为双曲线。曲线的弯曲趋势总是向大圆柱投影内弯曲。

基本绘图操作

偏移

偏移的命令为 offset ，可以将指定对象向指定的轴向方向移动。常用来绘制点划线和参考线。基本的操作步骤如下：

  1. 输入命令 offset ，回车确认。

  2. 输入偏移距离。[通过(T)/删除(E)/图层(L)]默认为<通过>。

  3. 选择需要进行偏移的对象。

  4. 指定要偏移的那一侧平面上的空白处。[退出(E)/多个(M)/放弃(U)]默认为<退出>。

  5. 回车确认绘制，退出。

修剪

修剪的命令为 trim ，可以删除掉指定的的绘图元素。基本的操作步骤如下：

  1. 输入命令 trim ，回车确认。

  2. 选择要修剪的对象。

基本图形元素的绘制

圆弧

绘制圆弧的命令为 arc 。如果中间不进行任何额外选择，默认是进行三点画圆弧。基本的操作步骤如下：

  1. 输入命令 arc ，回车确认。

  2. 指定圆弧的起点。

  3. 指定圆弧的第二个点。

  4. 指定圆弧的端点。

  5. 回车确认绘制，退出。

根据圆心进行绘制的基本操作步骤如下：

  1. 输入命令 arc ，回车确认。

  2. 根据[圆心(C)]输入 c ，回车确认。

  3. 指定圆弧的圆心。

  4. 指定圆弧的起点。

  5. 指定圆弧的端点。

  6. 回车确认绘制，退出。

在进行圆弧绘制的过程中，可以根据提示按住 Ctrl 切换圆弧方向。也可以根据[角度(A)/弦长(L)]进行选择来精确绘制确定长度的圆弧。

直线

绘制直线的命令为 line ，一般使用就是依次指定两个点确认即可。

如果要绘制指定长度和角度的直线，下面以绘制长度为60，与x轴夹角为120度的直线为例，其步骤如下：

  1. 输入命令 line ，回车确认。

  2. 指定第一个点。

  3. 输入 @60<120 ，回车确认。

  4. 按 ESC 退出绘制。

圆角

绘制圆角的命令为 fillet ，一般的使用步骤如下：

  1. 输入命令 fillet ，回车确认。

  2. 根据[放弃(U)/多段线(P)/半径(R)/修剪(T)/多个(M)]输入 r 回车确认。

  3. 输入圆角半径，回车确认。

  4. 选择第一个对象（边）。

  5. 选择第二个对象（边）。

圆

绘制圆的命令为 circle ，常常的使用情形时根据圆心和半径来绘制，其基本使用步骤如下：

  1. 输入命令 circle ，回车确认。

  2. 指定圆的圆心。[三点(3P)/两点(2P)/切点、切点、半径(T)]默认<圆心>。

  3. 输入圆的半径。[直径(D)]默认<半径>。
